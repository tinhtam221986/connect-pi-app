# CONNECT - Pi Network Developer Portal Guide

This guide helps you configure the **Pi Developer Portal** (develop.pi) to ensure your app works correctly on the Pi Browser.

## 1. App Configuration

*   **App Name:** CONNECT (or your chosen name)
*   **App Description:** A Web3 Social Video Platform on Pi Network.
*   **Category:** Social / Utilities

## 2. Connectivity Settings

*   **App URL:** `https://<your-vercel-domain>.vercel.app`
    *   *Note: This must be the HTTPS URL provided by Vercel.*
*   **Development URL:** `http://localhost:3000` (Optional, for local testing)

## 3. Privacy & Compliance (Critical for Mainnet)

The following URLs connect to the documentation generated by the app:

*   **Privacy Policy URL:** `https://<your-vercel-domain>.vercel.app/whitepaper`
*   **Terms of Service URL:** `https://<your-vercel-domain>.vercel.app/whitepaper`

*Note: We are using the Whitepaper page as the central compliance document as it contains the required Terms and Disclaimer.*

## 4. User Data Request (Scopes)

When the app asks for permissions, ensure you have enabled:
*   **username:** To display user profiles.
*   **payments:** To enable the Tipping and Shop features.

## 5. Hosting & Environment Variables (Vercel)

For the "Real User" Login to work, you **must** set the following Environment Variable in your Vercel Project Settings:

*   **Key:** `PI_API_KEY`
*   **Value:** (Your Pi API Key from the Developer Portal)

**How to get `PI_API_KEY`:**
1.  Go to `develop.pi` > Your App.
2.  Click **Generate New API Key**.
3.  Copy the string (starts with `Key...`).
4.  Paste it into Vercel > Settings > Environment Variables.
